# 부분 수열 찾기

### 자연수 n이 주어지면 1부터 n까지의 원소를 갖는 수열의 부분 수열을 모두 출력 (중복 허용 x)

입력: 3
출력: 
123
12
13
1
23
2
3


### 접근법  
n = 3, {1,2,3}

✅ 원소 하나도 부분 수열이다. ex) 1.   
✅ 부분수열 모두 출력 -> 방법의 수, 경우의 수와 관련이 있다.  
✅ 그렇다면 조합인지 순열인지를 고민해봐야 한다. 예를 들어 (1,3),(3,1)을 다르다고 표현하면 순서의 나열을 고려한 순열, 그렇지 않다면 조합. 문제에서 요구하는것은 조합.  
✅ 그렇다면 중복을 허용하는 중복조합인가? 예를 들어 222 가 허용?  
✅ 중복조합은 아니다. 그냥 조합.  
✅  N개의 숫자에서 한개를 나열하는 경우 + N개의 숫자에서 두개를 나열하는 경우 +N개의 숫자에서 3개를 나열하는 경우...N개의 숫자에서 N개를 나열하는 경우.   
✅ 이것은 결국 1부터 n까지의 모든 숫자를 돌면서 해당 노드의 숫자를 선택하느냐 안하느냐 로 이진트리를 완성할 수 있다.
